# 1 减速器的设计



## 1 设计题目

### 1.1 设计参数

**设计一级蜗杆减速器**

| 拉力$F$ | $2500N$  |
| :-----: | :------: |
| 速度$V$ | $1.6m/s$ |
| 直径$D$ | $450mm$  |

### 1.2 工作情况

每天工作小时数：16小时

工作的寿命：10年

每年工作：300天

三相交流电 ：380V/220V

### 1.3 减速器的设计步骤

1. 电动机的选择
2. 计算总传动比
3. 计算传动装置的动力学参数
4. **齿轮传动的设计(蜗杆传动)**
5. 滚动轴承的选择，传动轴的设计，校核
6. 键连接的设计选择
7. 联轴器的选型
8. 润滑
9. 热平衡计算
